What is this?

SmartPark is a carpark management web app that provides an easy way to manage your parking lot.

How is the code structured?

This is a Node/Express project. The file app.js is the server file that serves the routes. In the view directory are the html files, including index.html - our landing page - and about.html, which is the documentation (yup, that's this page).

The public folder holds the css and js files, as well as fonts and images that I use for this project. I will go into more technical details in the following section.

How is this app built?

As an overview, I used Node/Express as my base, and utilized Firebase as my databse. index.html includes all the visual elements, while script.js contains the logic. style.css contains the styles for everything. I also used D3.js for some data visualization.

SmartPark is designed to be structured as different pages. The landing page allows the user to initialize however many parking spaces he/she has in the carpark. The second page displays an overview of the carpark, namely the percentage of spaces occupied. The third page allows for some basic admin control over the carpark, including the logistics of a car entering and a car exiting. The date and time has to be entered in order to calculate the cost.
